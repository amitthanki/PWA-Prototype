Welcome to this sample PWA app!

This is a tech demo to show off what a PWA can do. If you're the latest version of Chrome,Firefox or Microsoft Edge you're in luck because they have support for PWAs. The demo is best to check out on your phone, unfortunately at this time PWAs are still a work in progress on iOS so we only have Android support.

Here's a rundown of some of the features of PWAs

  • Better UI experience (more app like)
  • Faster, because of offline caching
  • Homescreen icon installation support (similar to apps)
  • Offline Support
  • Push Notification
  • Many app like functionality in the works.

Technology that built this prototype

  • Angular 5
  • Angular Material Design
